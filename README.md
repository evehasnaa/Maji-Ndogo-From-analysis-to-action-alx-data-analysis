## 🚀 Week 3: From Basic Queries to Analytical Queries

This week marked a significant transition in the ALX Data Analytics course - we moved from basic SQL queries to advanced **analytical queries**.

### 📊 What We Learned

This was an intensive module covering 5 comprehensive chapters focused on:

- **Window Functions** - Advanced data analysis with OVER(), PARTITION BY, ROW_NUMBER(), RANK(), etc.
- **CTEs (Common Table Expressions)** - Writing more readable and maintainable complex queries
- **Pivot Tables** - Transforming data for better analysis and reporting
- **Data Manipulation** - Advanced techniques for data transformation
- **Text Transforming Functions** - String operations and text processing
- **Date Functions** - Working with temporal data effectively
- **Control Flow** - Conditional logic in SQL
- **CASE Statements** - Implementing complex conditional logic
- **IF Statements** - Simple conditional operations

### 💻 Technical Setup

We also learned how to integrate SQL with Python in a professional data science environment:
- Running SQL queries in **Anaconda environment**
- Working with **Jupyter Notebooks** for interactive data analysis
- Combining SQL with Python for powerful data workflows
